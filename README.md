# Halom Project

A distributed system project combining Substrate blockchain node, IPFS storage, and Python oracle services.

## Project Structure

```
halom/
├── node/               # Substrate node implementation (Rust)
├── oracle/            # Oracle service implementation (Python)
├── ipfs/              # IPFS node configuration and data
├── docker-compose.yml # Main orchestration file
└── .github/           # CI/CD configuration
```

## Components

- **Substrate Node**: Blockchain node implementation (placeholder)
- **Oracle Service**: Python-based oracle for external data integration
- **IPFS Node**: Distributed storage system

## Getting Started

### Prerequisites

- Docker and Docker Compose
- Python 3.8+
- Rust (for Substrate node)

### Running the Project

```bash
docker-compose up
```
